## Get Started

Required to install: [Node.js](https://nodejs.org/)

Clone Repository

```bash
git clone https://github.com/adydetra/rusun.git
cd rusun
```

Install package manager

```bash
bun install
```

Run Development

```bash
bun dev
```

---

> [!NOTE]
> if not using `bun` just delete `bun.lockb`, and switch to another package manager like:

```bash
# npm install
# yarn install
# pnpm install
```
